<p>MySQL query to extract domains from URLs:</p>
{% highlight sql %}
SUBSTRING_INDEX(SUBSTRING_INDEX(SUBSTRING_INDEX(SUBSTRING_INDEX(target_url, '/', 3), '://', -1), '/', 1), '?', 1) AS domain
{% endhighlight %}
